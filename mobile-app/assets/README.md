# Assets Folder

Esta carpeta debe contener:

- `icon.png` - Icono de la app (1024x1024px)
- `splash.png` - Splash screen (2048x2048px)
- `adaptive-icon.png` - Icono adaptativo Android (1024x1024px)
- `favicon.png` - Favicon web (48x48px)

Por ahora, la app funcionará sin estos archivos, pero se mostrarán placeholders.

Para crear los assets, ver `ASSETS_INSTRUCTIONS.md` en la raíz del proyecto.
